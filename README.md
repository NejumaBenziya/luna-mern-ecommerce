# 🛒 Luna - MERN E-commerce Application

## 🚀 Overview
Luna is a full-stack e-commerce web application built using the MERN stack.

## 🛠️ Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## ✨ Features
- Product listing
- Cart & wishlist
- User authentication
- Order management
- Admin & seller dashboard
